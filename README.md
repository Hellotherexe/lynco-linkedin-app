# Lyn&Co Realty LinkedIn AI Post Generator (n8n Workflow)

A fully automated n8n workflow that transforms screenshots of articles into formatted LinkedIn posts.

Upload your images, let AI extract and process the text, and review your generated posts in a custom user interface before publishing them directly to your personal LinkedIn profile.

## How It Works

1. **Upload:** User selects and uploads one or more screenshots of articles via a custom web interface.
2. **OCR Extraction:** The images are processed through the OCR.space API to extract the raw text.
3. **AI Generation:** The extracted text is grouped and sent to Google Gemini, which generates a tailored LinkedIn post for each image based on a specific prompt.
4. **Review & Edit:** The generated posts are displayed in UI cards where the user can read and edit the text.
5. **Approve & Publish:** With a single click, the approved text and its corresponding image are published directly to the user's personal LinkedIn profile.

> *Note: Currently, this workflow only supports posting to personal LinkedIn profiles, not Company Pages yet.*

## Setup Instructions

If you are importing this `linkedin-generator-workflow.json` into your own n8n instance, you will need to update a few things to get it working:

### 1. Import the Workflow
Download the `.json` file from this repository and use the **Import from File** feature inside your n8n canvas.

### 2. Add Your API Keys & Credentials
You will need to create and link your own credentials for the following nodes:
* **OCR Node (Code):** Open the code node named "OCR" and replace `YOUR_OCR_SPACE_API_KEY` with your free API key from OCR.space.
* **Google Gemini Node:** Add your Google Gemini API credential.
* **LinkedIn Nodes:** You must create an OAuth2 credential for LinkedIn. This requires setting up a developer app on LinkedIn to get a Client ID and Secret. Apply this credential to the **Get LinkedIn ID**, **Register Media Upload**, **Upload Image Binary**, and **Publish LinkedIn Post** nodes.

### 3. Update the Webhook URLs (If necessary)
The custom user interface relies on webhook URLs to process the uploads and approvals.
* Open the **Respond to Webhook (UI)** node.
* Look at the JavaScript `fetch()` calls near the bottom of the HTML code.
* Ensure that the paths (e.g., `/webhook/batch-processor` and `/webhook/approve-linkedin`) match the Production Webhook URLs generated by your n8n instance.

### 4. Customize the AI Prompt
Open the **AI Agent** node. The current prompt is specifically tailored for a luxury real estate agency. You will want to edit the prompt instructions, example posts, and required hashtags to match your own personal brand and writing style.
